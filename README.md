 
# PC Sales Data Aggregation (SQL Project)

## 📌 Overview

This project analyzes PC sales data using SQL aggregation techniques. It demonstrates how to extract meaningful insights from raw sales data by applying functions like `COUNT`, `SUM`, and `AVG`, combined with the `GROUP BY` clause.

## 🛠 Tools & Technologies

* SQL (Microsoft SQL Server)
* SQL Server Management Studio (SSMS)
* Visual Studio Code

## 📊 Key Features

* Total sales calculation using `SUM()`
* Average sales analysis with `AVG()`
* Record counting using `COUNT()`
* Data grouping by categories such as product, region, or time using `GROUP BY`

## 📂 Project Contents

* SQL scripts for data querying and aggregation
* Sample PC sales dataset
* Queries for grouped and summarized reports

## 🚀 How to Use

1. Open the project in VS Code or SSMS
2. Connect to your SQL Server database
3. Execute the SQL scripts
4. Review aggregated outputs for insights

## 📈 Objective

To practice and demonstrate SQL data aggregation skills for real-world business scenarios like sales reporting and performance analysis.
